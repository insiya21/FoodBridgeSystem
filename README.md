# 🍱 FoodBridge System

FoodBridge System is a full-stack food donation platform that connects Donors, NGOs, Volunteers, and Administrators to reduce food waste and help people in need.

## 🚀 Live Demo

Frontend:
https://food-bridge-system-git-main-insiya.vercel.app

Backend:
https://foodbridgesystem.onrender.com

---

## ✨ Features

### Donor

* Donate food with image upload
* View donation history
* Edit profile
* Change password

### NGO

* Claim available donations
* View claimed donations

### Volunteer

* View assigned deliveries
* Mark deliveries completed

### Admin

* Approve NGOs
* Assign volunteers
* Monitor donations

### Notifications

* OTP-based password reset
* Donation claimed notifications
* Delivery notifications

---

## 🛠 Tech Stack

* React.js
* Spring Boot
* Spring Security
* JWT Authentication
* PostgreSQL (Neon)
* Cloudinary
* Vercel
* Render
